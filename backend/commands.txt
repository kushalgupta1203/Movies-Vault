MOVIES VAULT - STARTUP COMMANDS
===============================

Follow these steps to start your Movies Vault application:

STEP 1: START DJANGO BACKEND SERVER
===================================

1. Open Terminal/Command Prompt
2. Navigate to backend directory:
   cd "d:\Projects\Movies-Vault\backend"

3. Activate Python virtual environment (if using one):
   # For Windows:
   venv\Scripts\activate
   # For Mac/Linux:
   source venv/bin/activate

4. Install dependencies (first time only):
   pip install -r requirements.txt

5. Run Django migrations (if needed):
   python manage.py makemigrations
   python manage.py migrate


6. Start Django development server:
   python manage.py runserver 8000

   âœ… Django server should now be running at: http://127.0.0.1:8000/
   âœ… Keep this terminal window open!


STEP 2: START REACT FRONTEND SERVER
===================================

1. Open a NEW Terminal/Command Prompt window
2. Navigate to frontend directory:
   cd "d:\Projects\Movies-Vault\frontend"

3. Install dependencies (first time only):
   npm install

4. Start React development server:
   npm run dev

   âœ… React app should now be running at: http://localhost:5173/
   âœ… Keep this terminal window open too!


STEP 3: ACCESS THE APPLICATION
=============================

1. Open your web browser
2. Go to: http://localhost:5173/
3. You should see the Movies Vault login page
4. Use any email/password to login (mock authentication)

TROUBLESHOOTING
===============

If Django server fails to start:
- Check if Python is installed: python --version
- Check if you're in the right directory: ls (should see manage.py)
- Try: python -m pip install django

If React server fails to start:
- Check if Node.js is installed: node --version
- Check if you're in the right directory: ls (should see package.json)
- Try: npm install (to reinstall dependencies)

If movies don't load:
- Make sure Django server is running on port 8000
- Check browser console for API errors
- Verify TMDB API key in backend settings
